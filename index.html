Workout Time Calculator

This webpage helps coaches and trainers calculate the duration of a workout program based on the number of sets, reps, and rest time. It will also provide an overview analysis of the difficulty of the workout based on basic factors.

How to Use

  1. Select the type of workout (For Time, AMRAP, EMOM).
  2. Enter the time cap in minutes, if applicable.
  3. Specify the number of rounds.
  4. Choose the workout type (Individual, Team Chipper, or Time-Based Team).
  5. Enter the team size, if applicable.
  6. Input the movements in the specified format, use the example as guidance.
  7. Click "Calculate Duration" to get the estimated workout time in the text below.
  8. For feedback or feature requests, use the feedback form at the bottom.
